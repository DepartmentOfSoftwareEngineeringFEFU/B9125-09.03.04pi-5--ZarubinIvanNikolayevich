void Circle() 
(Параметры запрашиваються у пользователя)
Вычисляет площадь круга по радиусу
Формула: S = π * R²
π ≈ 3.14
double radius = R - радиус круга

void Primoyugol()
(Параметры запрашиваються у пользователя)
Вычисляет площадь прямоугольника по длине и ширине
Формула: S = a * b
double dlina = a - длина прямоугольника
double shirina = b - ширина прямоугольника

void Paralelogram()
(Параметры запрашиваються у пользователя)
Вычисляет площадь параллелограмма по стороне и высоте
Формула: S = a * h
double storona = a - сторона параллелограмма
double visota = h - высота параллелограмма

void Romb()
(Параметры запрашиваються у пользователя)
Вычисляет площадь ромба по диагоналям
Формула: S = (d1 * d2) / 2
double diagonal1 = d1 - первая диагональ ромба
double diagonal2 = d2 - вторая диагональ ромба

void Sector()
(Параметры запрашиваються у пользователя)
Вычисляет площадь сектора круга по радиусу и углу
Формула: S = (π * R² * α) / 360°
π ≈ 3.14
double radius = R - радиус сектора
double angle = α - угол сектора в градусах (1-360)

void Trapecia()
(Параметры запрашиваються у пользователя)
Вычисляет площадь трапеции по основаниям и высоте
Формула: S = ((a + b) / 2) * h
double osnovanie1 = a - первое основание трапеции
double osnovanie2 = b - второе основание трапеции
double visota = h - высота трапеции

void Treugolnic()
(Параметры запрашиваються у пользователя)
Вычисляет площадь треугольника по трем сторонам
Формула: S = √(p*(p-a)*(p-b)*(p-c))
где p = (a + b + c) / 2 - полупериметр
double first = a - первая сторона треугольника
double second = b - вторая сторона треугольника
double third = c - третья сторона треугольника
Условие: a + b > c, a + c > b, b + c > a
